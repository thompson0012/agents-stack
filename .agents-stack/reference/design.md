# Design — agents-stack v3

## Visual Language

- File-first: all state concepts map to files and directories
- Phase hierarchy: each phase produces one markdown file

## Voice & Tone

- Precise, concise, engineer-to-engineer
- No flattery, no filler
- Chinese primary, English for technical terms
- Markdown for all durable artifacts

## Design Principles

1. **Files over chat** — durable artifacts are state, conversation is ephemeral
2. **Artifacts over assertions** — proof must be reproducible, not claimed
3. **Verification over agreement** — adversarial review catches self-deception
4. **Structure over improvisation** — defined phases prevent context drift
