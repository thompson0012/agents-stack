# Typography Terms Reference

Domain-specific terms for text-to-design prompt augmentation.

## Classifications

- **Serif** — Fonts with decorative strokes at stroke ends; traditional, readable body text
- **Sans-serif** — Clean stroke endings without flares; modern, minimal, UI-friendly
- **Display/font** — Decorative typefaces designed for headlines, not body copy
- **Monospace** — Uniform character width; used for code, data tables, terminals
- **Script** — Cursive or handwritten style; elegant, personal, formal invites
- **Slab serif** — Bold, block-like serifs; sturdy, vintage, editorial feel
- **Blackletter** — Gothic/medieval calligraphic style; dramatic, historical, ornamental

## Metrics

- **X-height** — Height of lowercase letters; affects readability at small sizes
- **Cap height** — Height of uppercase letters; defines vertical alignment
- **Baseline** — Line on which characters sit; anchors vertical rhythm
- **Ascender** — Part of lowercase letter above x-height (b, d, h); affects line spacing
- **Descender** — Part of lowercase letter below baseline (g, p, y); needs leading clearance
- **Tracking / Letter-spacing** — Uniform spacing across all characters in text
- **Leading / Line-height** — Vertical distance between baselines of consecutive lines
- **Kerning** — Adjustment of spacing between specific character pairs (AV, To)

## Hierarchy

- **H1** — Primary page heading; largest, highest visual weight
- **H2** — Section heading; second level of content organization
- **H3** — Sub-section heading; third level grouping
- **H4** — Detail heading; fourth level, often bold body-size
- **H5** — Minor heading; fifth level, subtle differentiation
- **H6** — Lowest heading; smallest, least prominent heading level
- **Display** — Oversized type for hero areas; not for reading flow
- **Headline** — Prominent text above body content; attention-grabbing
- **Title** — Named label for a section or component; medium emphasis
- **Subtitle** — Secondary text beneath a title; provides context
- **Body** — Primary reading text; optimized for legibility at 14–18px
- **Caption** — Small text describing images or media; low emphasis
- **Footnote** — Ancillary text at page bottom; supplementary info
- **Label** — Short identifying text for UI elements; scannable
- **Overline** — Small-caps text above a title; category or section marker

## Effect Terms

- **Drop shadow** — Offset shadow behind text; depth and separation
- **Outline** — Stroke around letterforms without fill; bold, graphic effect
- **Emboss** — Raised or recessed appearance; tactile, 3D illusion
- **Gradient fill** — Color transition across letterforms; vibrant, modern
- **All-caps** — Full uppercase transformation; assertive, formal
- **Small-caps** — Uppercase at lowercase height; refined, typographic
- **Ligatures** — Merged character pairs (fi, fl); elegant, reduces gaps
- **Tabular figures** — Uniform-width numerals; aligns in columns/tables
- **Oldstyle figures** — Varying-height numerals; blends into body text rhythm
