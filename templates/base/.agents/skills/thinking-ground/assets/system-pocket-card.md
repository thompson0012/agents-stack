# Three-Skill Pocket Card

Use this when the full system feels too heavy.

1. What am I protecting, escaping, or performing right now?
   - If the state is distorted -> `thinking-ground`
2. Can the problem be said in one sentence without smuggling in the solution?
   - If no -> `problem-definition`
3. Which 2-4 lenses actually matter here?
   - If yes -> `dynamic-problem-solving`
4. If I am wrong, what is the most likely reason?
   - Use this before committing

## Exit reminders

- simple -> apply the known practice
- complex -> run small experiments
- chaotic -> stabilize first
- hollow -> less analysis, more reality contact
