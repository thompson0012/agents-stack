# Lessons

## Mistakes

- Treating template live-doc placeholders as if they were active repo state creates false continuity after copy.
- Letting required local guidance exist outside the indexed hierarchy makes the template harder to use correctly.

## Anti-Patterns

- Hidden must-read local guides.
- Non-durable local guides added at leaves, utility folders, or other boundaries with no distinct copied contract.
- Blurring shipped and optional skill-package truth in the same inventory.

## Fixes

- Keep the root guide short, constitutional, and explicit about every must-read local guide.
- Add local guides only at durable copied boundaries where rules genuinely diverge from the parent.
- Keep template live docs inert until localization, and keep shipped and optional skill-package surfaces separate.
