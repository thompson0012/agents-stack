# Live Docs Guide

This `AGENTS.md` extends `../AGENTS.md`. Root constitutional rules cannot be overridden.

## Local Scope

These files are runtime-state scaffolds for copied repos. In the template, they must remain inert and must not be treated as active project state.

## Rules

- Do not treat placeholder labels as active repo state.
- Do not seed fake objectives, progress, blockers, or handoff notes.
- Localize these files after copy before relying on them as truth.
- `roadmap.md`, `runtime.md`, and `qa.md` may remain planned or unused in a downstream repo until that workflow is activated.

## Required Reads

- Read this file before relying on `current-focus.md`, `progress.md`, `todo.md`, `roadmap.md`, `runtime.md`, or `qa.md`.
